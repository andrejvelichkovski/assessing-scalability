Welcome

This webpage is used for benchmarking purposes as part of the "Assessing the Scalability of Lightweight Virtualization Systems".

More information about the project can be found on the GitHub project page

Benchmarking process

This is a static page that will be hosted on a Nginx server. This Nginx server will be run either in a Docker environment or in an Unikraft uni-kernel virtual machine.

Apart from this type of benchmark, there are other types of benchmarks performed. Check the GitHub repository above to find out more about them.
